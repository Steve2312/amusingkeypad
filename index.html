What is it?

Switches

We sell the keypad with Cherry MX switches. The availiable switches are MX Blue, MX Red, MX Black, MX Brown and MX Silver.

Key bindings

With the drivers you can bind any number of normal key values to the same button to achieve the same input. The drivers will store the new configuration in the keyboard chip, so even if you connect it to other devices, it can be used.

Lighting modes

It contains multiple rgb-leds, and you can specify different colors or display modes for each Led.
